# 🍕 AI 기반 피자 타이쿤 게임: 좋은 피자, 위대한 피자

## 프로젝트 개요

이 프로젝트는 Ollama의 EEVEE 모델을 활용하여 AI 기반 피자 타이쿤 게임을 제작한 프로젝트입니다. 게임 내에서 AI가 손님 역할을 맡아 다양한 까다로운 주문을 처리하고, 사용자는 이를 바탕으로 피자를 제작하여 게임을 진행합니다. AI는 다양한 주문 옵션을 이해하고, 그에 맞는 피자를 만드는 과정에서 플레이어에게 도전 과제를 제공합니다.

### 주요 기능:

- AI 손님: EEVEE 모델을 사용하여 다양한 주문을 생성하고, 플레이어에게 도전적인 주문을 요구

- 피자 제작: 플레이어가 손님의 요구에 맞는 피자를 만들어 제공

- 게임 진행: 주문을 처리하고, 피자를 정확히 만드는 능력에 따라 점수와 피드백 제공

## 게임 규칙

게임에서 손님은 매우 어렵고 복잡한 피자 주문을 합니다. 손님의 주문은 종종 직관적이지 않으며, 여러 가지 요소가 섞여 있어서 플레이어가 정확하게 주문을 해석하고 맞춰야 합니다.

- 예시:

    "페퍼로니랑 소세지랑 사귀는 거 맞죠?"

이 주문은 도우, 치즈, 소스, 페퍼로니, 소세지가 포함된 피자를 뜻합니다.

플레이어는 손님의 요구를 정확히 이해하고, 그에 맞는 피자를 만들어야 합니다.

플레이어는 손님의 요구사항을 정확하게 반영하여 피자를 준비하고, 이를 제공하는 방식으로 게임을 진행합니다.

## 점수 시스템

게임에서 점수는 보유 금액으로 결정됩니다. 게임을 시작할 때 금액은 0원으로 시작하며, 피자 주문을 처리하여 금액을 벌거나 잃게 됩니다.

- 이익 계산

    이익은 주문을 성공적으로 처리했을 때와 실패했을 때 각각 다르게 계산됩니다:

    - 성공적인 주문 처리:

        가격에서 재료비를 뺀 금액이 보유 금액에 추가됩니다.

    - 주문 실패:

        주문을 실패하면 재료비만 차감됩니다.
